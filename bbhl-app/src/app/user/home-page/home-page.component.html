Welcome to the BBHL Angular App

This app has no affiliation with the real BBHL..... it is simply an excuse for me to learn some Angular. That being said, all the stats and results shown herein actually happened!

  • Try my Draft Simulator!

    This is why you're here. Pick your BBHL franchise, randomize a draft order, and start picking. This fully functional simulator includes a dashboard to help you sort through the available players and visualize your lineup as you draft.

    Start drafting
  • Play some BBHLdoku

    It's no draft sim but this game is pretty cool too! Think you know everyone in league? Prove it in this PuckDoku inspired guessing game. Name players who've bounced around the league and get rewarded for more obscure guesses. Please play it so it can learn from your guesses.

    Play now
